---

## ⚡ DÉPLOIEMENT EN 5 MINUTES

### **ÉTAPE 1 : Créer un repo GitHub** (1 min)

1. Va sur **github.com** → Sign in
2. Clique **"New"** (en haut à droite)
3. Appelle-le `maison-delvaux`
4. Clique **"Create repository"** ✅

---

### **ÉTAPE 2 : Ajouter les fichiers** (2 min)

**OPTION A : Upload ZIP (Plus rapide)**
1. Dans le repo GitHub, clique **"Add file"** → **"Upload files"**
2. Drag & drop le dossier `maison-delvaux/` complet
3. Message : "Initial commit"
4. **Commit changes** ✅

**OPTION B : Copier fichiers un par un**
1. **Add file** → **Create new file**
2. Pour chaque fichier en haut, crée-le :
   - `package.json`
   - `vercel.json`
   - `.gitignore`
   - `README.md`
   - `public/index.html`
   - `src/index.js`
   - `src/App.jsx` (copie le contenu de VERSION_2_STONE_GOLD.jsx)

---

### **ÉTAPE 3 : Déployer sur Vercel** (2 min)

1. Va sur **vercel.com** 
2. **Sign Up** (avec GitHub si possible)
3. Clique **"Import Project"**
4. Sélectionne `maison-delvaux` depuis GitHub
5. **Deploy** 🎉

**Boom !** T'as une URL de démo live avec photos + vidéo !
